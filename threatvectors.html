Potential Threat Vectors for MJ Residence

Threat Vector Description MITRE ATT&CK Tactic MITRE ATT&CK Technique Mitigation
Phishing Attack Attackers send deceptive emails to trick users into revealing sensitive information. Initial Access Phishing (T1566) Implement email filtering and educate users on recognizing phishing attempts.
Brute Force Attack Attackers attempt to gain access by systematically guessing passwords. Credential Access Brute Force (T1110) Enforce strong password policies and implement multi-factor authentication (MFA).
Malware Infection Malware is introduced to the network via infected email attachments or malicious websites. Execution Malicious File (T1204.002) Deploy antivirus software and regularly update all systems and software.
Unauthorized Network Access Attackers gain unauthorized access to the home network through unsecured devices. Lateral Movement Exploitation of Remote Services (T1210) Secure all devices, change default passwords, and use network segmentation.
Data Exfiltration Attackers steal sensitive data from the network and send it to an external location. Exfiltration Exfiltration Over C2 Channel (T1041) Monitor network traffic and use data loss prevention (DLP) tools.
Physical Intrusion Unauthorized physical access to the residence to steal or tamper with devices. Impact Physical Theft (T1070) Install surveillance systems and secure entry points.
